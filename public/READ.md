# Fatty House 🍔
Fast-food restaurant websit built with HTML, TailwindCss and JavaScript.

## Demo Link
[Live Demo](link)

## Features
- Responsive Website both on mobile and Large Screen
- Navbar with mobile menu links and scroll effect
- Hero section with background image and CTA button
- About us section
- Featured Menu Section with Featured Menu card and CTA button
- Contact form with address, phone and email and send message form
- Footer with quick links & social icons

## Techologies
- HTML
- TailwindCSS
- JavaScript
